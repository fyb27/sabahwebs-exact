# sabahwebs.com — exact static clone

A faithful, self-contained static mirror of the live site **https://sabahwebs.com/**
(11 pages: homepage + 10 blog posts), captured 2026-06-09.

All CSS, JavaScript, fonts, and images are self-hosted under this repo — the site
renders without any dependency on the original Webflow CDN. Outbound links and SEO
canonical/og tags still point to the original domain.

Served as static files via GitHub Pages (`.nojekyll`).
